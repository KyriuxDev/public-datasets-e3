# Stats NZ — Large Datasets

Colección de datasets de gran volumen publicados por Statistics New Zealand (Stats NZ), el organismo oficial de estadística del gobierno de Nueva Zelanda. Los archivos están disponibles en formato CSV para descarga directa.

## Diccionario de Datos

| Columna | Tipo | Descripción |
| :--- | :--- | :--- |
| `dataset_name` | `String` | Nombre del conjunto de datos |
| `category` | `String` | Categoría temática (ej. población, economía, medio ambiente) |
| `year` | `Int` | Año de referencia del registro |
| `region` | `String` | Región geográfica de Nueva Zelanda |
| `value` | `Float` | Valor numérico del indicador medido |
| `unit` | `String` | Unidad de medida del valor (ej. personas, dólares NZD) |
| `frequency` | `String` | Frecuencia de actualización (anual, trimestral, mensual) |

## Fuente

[stats.govt.nz — Large Datasets CSV](https://www.stats.govt.nz/large-datasets/csv-files-for-download/)
